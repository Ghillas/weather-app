# Weather App

Check the current weather on any city on the planet.

<img width="1254" height="595" alt="screen_interface" src="https://github.com/user-attachments/assets/ff6f434d-8a43-45d8-b753-f684f1d172c1" />


## Features

1. Current time and date of the city in the configuration file

2. Temperatures and humidity

3. Wind speed and direction

4. Sunrise and sunset times

## Installation

1. `git clone https://github.com/Ghillas/weather-app.git`

2. `cd weather-app`

3. `npm install`

4. `npm run dev`

## Change city configuration

1. Found the configuration/city.json file

2. Change the city and country names as well as the geographic coordinates to those of the city of your choice
